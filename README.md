# 打造餐廳清單－實作練習
使用 Node.js & Express （handlebars）方式製作餐廳美食清單

# 功能
．搜尋餐廳
．點擊餐廳查看細項
．點擊我的餐廳清單回到首頁

#  複製或下載此專案
    git clone https://github.com/carlos811009/Restaurant-website
 
## 本地安裝 Node (Mac)
### 使用terminal ###
`$ curl -o- https://raw.githubusercontent.com/creationix/nvm/v0.34.0/install.sh | bash`

### 檢查是否安裝成功 ###
`$ command -v nvm`

# 執行專案 #
### 使用terminal ###
`$ node app.js`
